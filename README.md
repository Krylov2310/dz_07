# Итоговые пояснения:

## 01 HashTable

- Реализует хеш-таблицу с линейным пробированием.
- Автоматически расширяется при заполнении на 70%.
- Поддерживает основные операции: вставка, поиск, удаление.
---
## 02 string_hash

- Простая хеш-функция на основе суммы ASCII-кодов.
- Не является криптографической, но подходит для учебных целей.
---
## 03 StringHashDict

- Словарь, использующий хеш-функцию для ключей-строк.
- Хранит как значение, так и хеш-код ключа.
- Обеспечивает удобный интерфейс для работы с парами ключ‑значение.
